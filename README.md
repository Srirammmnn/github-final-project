 Simple Interest Calculator

## 📌 Project Overview

This repository contains a full stack application for calculating simple interest. It demonstrates the integration of frontend, backend, and scripting components in a single project.

The application calculates simple interest using the formula:

Simple Interest (SI) = (Principal × Rate × Time) / 100

---

## 🧠 Project Description

The project is designed to help users easily compute simple interest by entering:
- Principal amount
- Rate of interest
- Time period

It includes:
- A **Shell Script** for command-line usage
- A **Flask Backend API** for processing calculations
- A **Frontend Interface** using HTML, CSS, and JavaScript

---

## ⚙️ Features

- Simple and fast calculation
- Clean user interface
- API-based architecture
- Command-line support
- Beginner-friendly full stack project
